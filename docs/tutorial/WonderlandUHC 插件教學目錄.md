# WonderlandUHC 插件教學目錄

歡迎閱讀本插件的教學文檔！我們希望用最淺顯易懂的教學，讓所有人都輕鬆的學會如何架設伺服器，與玩家們一起享受UHC比賽！

## 注意事項

1. 本教學主要提供「完全沒有開過伺服器，想要嘗試成為UHC主持人」的使用者閱讀，因此會以「開一個純WonderlandUHC伺服器」作為教學目標，若您是熟悉經營伺服器的使用者，歡迎指出本教學的錯誤，但請不要批評為了面向初學者而精簡的教學內容。

2. 架設伺服器屬於Minecraft裡具有一定學習門檻的技術，**若您具備基本的英文閱讀能力或資訊領域相關知識，能夠大幅減輕除錯的時間與成本**。

3. 如果您是對於這方面一竅不通的使用者，請具備足夠的耐心學習，並且 **操作時請不要過於害怕！** 本教學會非常仔細的敘述每一項步驟，確保所有前來閱讀的使用者都能順利開啟伺服器！

4. 由於每個人的電腦與網路環境皆有非常大的差異，本教學已盡可能列出多數人會遇到的問題與解決方式，但仍然可能漏掉許多狀況，因此在學習開服時，請盡量**確保自己擁有足夠現代的電腦硬體與網路環境。**
