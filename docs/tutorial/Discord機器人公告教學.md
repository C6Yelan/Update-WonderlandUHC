# WonderlandUHC Discord機器人公告教學
若您身為Minecraft社群管理者，肯定希望有一個便利的方式宣傳UHC比賽，WonderlandUHC將此也納入插件的一部份，透過連動DiscordSRV插件，實現此功能！

## 事前準備
本步驟需要您擁有一個已安裝WonderlandUHC插件，並預載入完成的伺服器，若你還沒有做好準備，請參考上一部[WonderlandUHC主持教學](UHC主持教學.md)。
